INTRODUCCIÓN A PHP

PHP (acrónimo recursivo de PHP: Hypertext Preprocessor) es un lenguaje de programación de código abierto, ampliamente utilizado especialmente para el desarrollo web del lado del servidor.

Características principales de PHP:
  • Se ejecuta en el servidor web y genera contenido HTML dinámico que se envía al navegador.
  • PHP puede incrustarse directamente en el código HTML, lo que facilita la creación de páginas web interactivas.
  • PHP se integra fácilmente con sistemas de gestión de bases de datos como MySQL, PostgreSQL, SQLite, entre otros.

Como Ejecutar Codigo PHP

Para ejecutar código PHP es necesario preparar un entorno que incluya un servidor web y el intérprete de PHP.

Algunas opciones para descargar estos recursos son (XAMPP) (LARAGON), (MAMP), entre otros.

Para el desarrollo de este curso utilizaré la herramienta (XAMPP), como instalador de los recursos que se necesitan para poder ejecutar código php.

Tutorial de instalación: XAMPP

XAMPP incluye el servidor web apache, una version actual del interprete de php, y el motor de base de datos MySQL. (También incluye otras cosas más, pero no serán necesarias de momento).

Crear Un Proyecto PHP

Una vez instalado (XAMPP), hay q seguir los siguientes pasos:

  1. Entrar a la carpeta (xampp) que se crea al instalar (XAMPP).
  2. Entrar a la carpeta (htdocs).
  3. Crear una carpeta con el nombre del proyecto.
  4. Dentro de la carpeta con el nombre del proyecto, ya se podrán crear todo tipo de archivo que necesites para el proyecto.

NOTAS

  • No olvidarse inicializar el servidor para poder visualizar lo codigicado.

  • Indicar correctamente la ruta del archivo en el navegador, para poder acceder a este.

  • Para poder escribir codigo php, es necesario escribirlo dentro de las estiquetas (etiqueta de apertura:<?php) y (etiqueta de cierre: ?>). Ejemplo:

    
                <?php
                    echo "Este Es Código PHP.";
                ?>
  • Si es un archivo en el que solo se escribirá codigo php no hace falta poner la etiqueta de cierre, pero si se conbina con codigo html, si hace falta.
